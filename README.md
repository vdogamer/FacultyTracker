# FacultyTracker

FacultyTracker is a training application for learning C# ASP.NET Core MVC and SQL Server 2019.

## Purpose

This project documents my progress in building a basic web-and-database application from the ground up.

## Training Goals

- Learn ASP.NET Core MVC fundamentals
- Connect a web application to SQL Server 2019
- Build CRUD screens
- Practice C# models, controllers, views, and services
- Improve SQL querying and database design skills
- Add validation, search, filtering, and reporting
- Document progress for professional development and performance review evidence

## Business Scenario

The application uses a sample faculty profile tracking scenario.

The app will eventually track:

- Faculty name
- Email
- Rank/title
- Department
- Active/inactive status
- Created and modified dates

## Technology Stack

- C#
- ASP.NET Core MVC
- SQL Server 2019
- Entity Framework Core
- Visual Studio
- GitHub
